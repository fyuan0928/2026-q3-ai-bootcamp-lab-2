## Running Tests

All commands below are run from the root of the repository.

| Command | Description |
|---------|-------------|
| `npm run test:frontend` | Run frontend unit tests (Jest + React Testing Library) |
| `npm run test:backend` | Run backend unit tests (Jest + Supertest) |
| `npm test` | Run frontend and backend unit tests |
| `npm run test:integration` | Run backend integration tests |
| `npm run test:e2e:install` | Install Playwright's Chromium browser (first-time setup) |
| `npm run test:e2e` | Run end-to-end tests (Playwright) |
| `npm run test:all` | Run unit, integration, and E2E tests |

See [Testing Guidelines](docs/testing-guidelines.md) for conventions on writing and naming tests.
